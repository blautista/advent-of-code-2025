# Solutions to advent of code 2024 in typescript

## How to run:

* `npm i`.
* Create a folder named `inputs` on the root. 
  * Each puzzle input should be in a file named `${{DAY}}.txt`. 
  * Each test input should be in a file named `${{DAY}}_test.txt`
* `npm run day -- {{day}}`. For example, to execute day 5 do ``npm run day -- 5``.

## WARNING

I'm not a leetcode dude. These are not the most efficient or simple solutions, but they work. 

No llms used, these are solved just for practice.
