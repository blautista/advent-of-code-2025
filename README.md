# Solutions to advent of code 2024 in typescript

## Run:

- Install deno.
- Run a solution with `deno task day {{day}}`. For example, to run day 5 do `deno task day 5`.

Each puzzle input should be in a file named `input.txt` within the day folder.

Each test input should be in a file named `test_input.txt` within the day folder.

## Add a day:

- Run `deno task create-day {{day}}`.

## WARNING

I'm not a leetcode dude. These are not the most efficient or simple solutions, but they work.

No llms used, these are solved just for practice.
